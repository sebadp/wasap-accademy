## Qué es AgentCraft

AgentCraft es una plataforma gamificada para aprender **Inteligencia Artificial Generativa** de la manera más efectiva posible: construyendo un proyecto real de producción.

<Callout variant="tip" title="La filosofía">
No aprendas de ejemplos de juguete. Aprende entendiendo, modificando y extendiendo código de producción real.
</Callout>

El proyecto que vas a estudiar se llama **LocalForge**: un asistente de WhatsApp y Telegram con IA que tiene 85 archivos Python, 15+ skills, 316 tests, y sistemas completos de memoria, RAG, tool-calling, agentes y trazabilidad.

<Concept title="LocalForge">
LocalForge es un asistente personal que corre en tu propia máquina. Usa **Ollama** como motor de LLM (modelo qwen3.5:9b para chat y llava:7b para visión), **SQLite** como base de datos, **FastAPI** como framework web y **faster-whisper** para transcripción de audio. Todo es local y open-source: no dependés de APIs pagadas.
</Concept>

## El stack tecnológico

LocalForge no es un chatbot sencillo. Es un sistema de producción con múltiples subsistemas integrados. Este es el stack completo:

| Componente | Tecnología | Para qué |
|------------|-----------|----------|
| Framework web | FastAPI (async) | Recibir webhooks, servir la API |
| LLM (chat) | Ollama + qwen3.5:9b | Generar respuestas inteligentes |
| LLM (visión) | Ollama + llava:7b | Analizar imágenes enviadas por el usuario |
| Audio | faster-whisper | Transcribir mensajes de voz |
| Base de datos | SQLite + aiosqlite | Conversaciones, memorias, notas |
| Búsqueda semántica | sqlite-vec + nomic-embed-text | RAG sobre memorias y notas |
| Embeddings | nomic-embed-text (768 dims) | Vectorizar texto para búsqueda |
| Mensajería | WhatsApp Cloud API + Telegram Bot API | Comunicarse con los usuarios |
| Observabilidad | Langfuse v3 | Tracing y evaluación de calidad |

<Callout variant="info" title="Todo local, todo tuyo">
A diferencia de otros asistentes que dependen de OpenAI o Anthropic, LocalForge corre 100% en tu máquina. Ollama ejecuta los modelos localmente, SQLite almacena todo en un archivo, y faster-whisper transcribe audio sin mandar nada a la nube.
</Callout>

## Lo que hace a LocalForge único

No es otro tutorial que te enseña a llamar a la API de OpenAI. LocalForge tiene sistemas de producción reales:

<Concept title="Tool Calling con Router de 2 etapas">
El LLM (qwen3.5:9b) solo puede manejar ~5-6 tools de forma confiable. Pero LocalForge tiene 60+ tools. La solución es un **router de 2 etapas**: primero un LLM rápido clasifica la intención del usuario, y luego se seleccionan solo las tools relevantes (máximo 8) para esa categoría.
</Concept>

<Concept title="Memoria avanzada con consolidación">
LocalForge no solo guarda lo que le decís. Tiene daily logs, session snapshots, flush de conversaciones a memoria permanente, deduplicación con SequenceMatcher, y consolidación periódica via LLM para eliminar memorias duplicadas o contradictorias.
</Concept>

## Prerequisitos

Para aprovechar al máximo AgentCraft, necesitás:

- **Python 3.11+** — el proyecto usa type hints modernos (`list[str]`, `X | None`)
- **Conocimiento básico de async/await** — FastAPI es 100% async
- **Docker Desktop o Docker Engine** — para levantar Ollama y servicios
- **8GB+ de RAM** — los modelos de Ollama necesitan memoria

<Callout variant="warning" title="No necesitás ser experto">
Si sabés hacer un `for` loop en Python y entendés qué es una API REST, estás listo. Todo lo demás lo aprendés en el camino.
</Callout>
